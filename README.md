Answer the question no 1:
getElementById() is used to find single element
getElementByclassName() is used to find many element by class

querySelector is used to find the first element by css selector
querySelectorAll is used to find all elements by css selector

Answer the question no 2:
A new element is created using createElement() and it is added to the DOM using appendChild().

Answer the question no 3:
Event Bubbling means an event moves from a child element to its parent element.

Answer the question no 4:
Event Delegation means using one event listener on a parent element to handle events of its child element.


Answer the question no 5:
preventDefault() Stops the browser’s default action.
stopPropagation() Stops the event from going up to parent elements
